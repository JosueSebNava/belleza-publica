# Belleza Pública 💅✨

Una aplicación web full-stack para la gestión y visualización de servicios de belleza.

---

## 🚀 Características

- **Frontend:** Interfaz dinámica e interactiva construida con HTML, CSS y JavaScript.
- **Backend:** API REST desarrollada con Node.js y Express.
- **Autenticación:** Sistema de usuarios/autenticación integrado.
- **Base de datos:** Almacenamiento dinámico de datos (`JSON` / base de datos local).

---

## 🛠️ Tecnologías utilizadas

- **Backend:** Node.js, Express
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Despliegue:** Render / Vercel

---

## 💻 Instalación y ejecución local

Sigue estos pasos para ejecutar el proyecto en tu entorno local.

### 1. Clonar el repositorio

```bash
git clone https://github.com/TU_USUARIO/belleza-publica.git
cd belleza-publica
```

### 2. Instalar las dependencias

```bash
pnpm install
```

### 3. Iniciar el servidor de desarrollo

```bash
pnpm start
```

La aplicación estará disponible en:

```text
http://localhost:3000
```

> **Nota:** Si configuraste un puerto diferente mediante una variable de entorno, utiliza ese puerto.

## 🌐 Despliegue

La aplicación está optimizada para desplegarse fácilmente en **Render**.

### Configuración recomendada

| Opción | Valor |
|---------|-------|
| **Build Command** | `pnpm install` |
| **Start Command** | `node server.js` |

### Pasos

1. Conecta este repositorio a tu cuenta de Render.
2. Configura el comando de compilación:
   ```bash
   pnpm install
   ```
3. Configura el comando de inicio:
   ```bash
   node server.js
   ```
4. Despliega la aplicación.
